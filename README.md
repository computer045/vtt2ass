# VTT to ASS

This is a simple application to convert VTT files to ASS subtitles.

## Requirements
- ruby 2.7.1 or newer

## Prerequisites

This application requires the use of the `os` gem.

To install:
```bash
gem install os
or
bundle install
```

## Usage

Add your VTT files in the `input` directory.

Run this command:
```bash
$ ruby main.rb
```
